社会学就业探讨

发表于 2020-10-09 更新于 2022-02-27 本文字数： 2k 阅读时长 ≈ 2 分钟

想法链接：

https://www.zhihu.com/pin/1297698449555959808
https://www.zhihu.com/question/424735448

Body:
我觉得吧，原回答你们看两点就可以了

  1. （大多数该专业的人）不好就业
  2. 社会学学界正在探讨的问题，相较其他学科都相对过时了

回答这个问题根本不需要任何的社会学基础，只不过两个人越说越复杂了，我直接用大白话讨论一下

别扯谁上没上过课，会不会critical thinking（反正你们以后当了AP有的是时间吵去…），我就这么问：

第一点：5年之内我们这个项目组一共能申请到多少经费，我们组薪金最高和最低的，各自能分到多少？

没钱谁都玩不转，虽然肯定不至于饿死，但只要钱少到让你感觉到【不值】，你基本上就很难再坚持下去了，尤其是等你们有了家庭之后钱的开销会越来越大的…

针对大家可能对第一点进行的反驳（经费什么的是根据项目实际开展情况动态调整的，不可能一成不变），我就进一步问，

第二点：我们能不能（能的话要怎么样）通过（除了发表论文以外的）其他方式来增加申得经费的总量？人数净增长大家都知道，那么这样我们怎么保证末端的学术从业人员的待遇，能够随着行业的发展而保持平稳（或者至少不能滑坡得太厉害）？

不要简单地认为（船到桥头自然直），这个逻辑就跟以前人们常说的【多生个孩子就是多双筷子】一样…

针对大家可能对第二点进行的反驳（我们确实只是产出论文的机器，但是我们啊是实实在在推动了社会学这个行业在进步，也密切关注着社会里各种要素随着时间的兴衰更替…），那么我就第一步问，

第三点：我们凭借什么来（或者让人们相信你们正在（或最终能够））体现自己这个学科的贡献与作用？社会学的哪些研究，能够让人们（至少得让大家的金主们）觉得这个问题很值得探讨、很有
研究价值？
换句话说：
我们能这堆破论文中提取出哪些新问题？这个问题的回答，是不是也就能部分地解答【学界前沿到底有多大的发展前景】了？直观地这么想，我们在关心的东西，与金主们也在关心的问题，两者是不能完全没有交集的…
Link: 如何评价陈健坤关于“博士最惨能惨到什么程度”一问的回答及关于社会学的相关争论？

Comment:
我把话说清楚，这就是理想与不理想的问题，这就是科研产业化必然要面对的情景，每个人都作为一个datapath一样的管子，粘在两个接头的两端，然后凭借着这个所处的位子接受自上或自下的供奉…

你可以不想做，也可以不做，但不做你就得一边工作一边搞科研（当然一些聪明的脑袋瓜马上就会开动起来，我们不生产知识，我们只是大量廉价科研劳动力的搬运工，我们借着宣传，招摇过市，让他们明白星辰大海总在等着我们，他们就会自动忽略掉所有的前路茫茫…）

总之你肯定是要有钱才能生活下去的，那么当你不想当管子、想要【做自己的研究】的时候，你就得面临赚不到钱然后饿死的风险，也就是说，（因为你这辈子不曾且终将无法财务自由，你又想体面地过着平静的生活，所以这个研究的方向就麻烦先生您等待时机，等到有缘的弟子再行探讨，再要不就等到下辈子吧…）
马克思这么厉害的社会学家（他自己不承认但我们都知道他是），在飞利浦和恩格斯的接济下尚能如此潦倒，大家自己盘算咱们中到底几个有像飞利浦恩格斯这样的亲友能让你们这样借钱？（孔德就不说了，这个偏差值大了些…）
研究社会学的个体同样作为某一类社会（并且往往也是他所在的那个社会）的一项组成，并且其行为可能发挥相当显著的效用（马克斯韦伯同学，抽到你了请起立），这样一来他的观察可能因此受到一定影响，二来他自己也只能基于他所在的那一两百年进行观察，这样观察到的只是特定历史场景下的社会状态，延续性不一定足够…

算了，直接上暴论，累了（

所有当前在世的人类（假设都不超过120岁），都不是自主选择了1900-2020年这个span下任何一种或一类社会状态之后才出生的，但是我们谈到社会的时候竟然丝毫感觉不到自身【在场】这一现象的荒诞感，一般人也自然而然地认为社会是【不完美现实混合叠加下的特定时空状态】或者【各类人群发出噪音、展示丑态、渴求真善美的世界规模的集体舞会】，这个事实也一点不令他们感到惊讶…

如果说现代性只是带来诅咒，那么构成【魔域】的是物化的人与监视着全部物化的市场，这个魔域缢裂了原生的社会结构，悬临于星射线上空的是【取得了最佳配置方式】的人一遍遍地被神格化，跌入底层的弱势者被迅速资源化形成养料，他们徒然而坚忍地走着，走向那个发放【发展必然的阵痛】牌子的检疫印章的屠宰场

大家都已经活得足够撕裂了，哪里还有心思去发放善意给这些【易碎的绝对】啊…
我的存活不过基于一点执念，也即就是不想让他们这么诗意地【在】着，才要打碎这一切的；就是渴望有人能终结这横亘于60亿人口上方的巨型格式塔，才要等待到哪怕大厦崩塌的；就是为了给够哪些在实在界里饥渴濒死的魂灵以毒药一般苦涩而致命的希望，才要和他们联手，一同幻化为林林总总的海市蜃楼.
