剧透警告

人们一般会认为，只浏览到前几回的玩家根本没有说剧透警告的必要，但我隐约感觉我已经快猜出这个故事的走向了，所以为了以防猜想成真、变成一个潜在的spoiler，这里就先做一个预警

我看书或者动漫有个一贯的先行习惯，即未读先猜，猜一下后续的剧情或象征，因此

以下均为片面的猜测与解读

人物取名

顾韦、顾真真 = 真伪兄妹（
一来是兄妹只是惯称而非血缘，
二来可能之后真伪会有对抗（吵架？会因为什么而吵呢？），
三则是“假作真时真亦假”，伪物往往是比真物还要可贵的东西…
）
钟齐北 = 钟馗or汉钟离 （想都知道，一个道士偏偏姓钟，这是很不得了的一件事，在古代“钟”即“律”即“神”）
苏半夏 = （半夏属于中药，属性是木中之土（土在内是不好的象征，表示内心郁结…ntmd谁会用中药去给女孩子命名啊，滚滚滚） -> 标准女主用名（不存在的夏天，大概也就是苏半夏泳池话之后消失了（并且估计没人记得，老志仓千代丸了）所以叫“不存在的夏天”）

苏溯 = （溯即逆流，小兄弟的生活可能不大如意…滚滚滚） -> 姓苏只是表示跟苏半夏是姐弟

梦野瞳 = 瞳这里即（异瞳），也就是苏半夏（这个ch0就告诉你了好吧）
（注意这边梦野瞳的形象是红中透蓝，不用多想肯定是反色致敬珂朵莉啦…gun）

PV

泪痣

从PV上看，很容易看出苏半夏和苏溯右眼下都有泪痣（泪痣貌似不能遗传吧），显然不是巧合，所以第一个猜想是：苏半夏跟苏溯应该是同一个人
（我猜大概是其中一方，即姐姐把弟弟杀了，然后以另一个人的身份，必要时出现在男主周围，目的是从男主身上搞到什么东西，这种东西使男主得到强制平凡，然后苏半夏泳池话里不正表示想当个平凡的人嘛——园山赤音表示wdnmd）（异瞳可以带美瞳解决，不难处理）
（但姐姐的头发该怎么解释？胸又怎么解释？所以以上的假设是很不靠谱的：毕竟胸是很难藏起来的，假发在泳池里也必然暴露）
但泪痣毕竟不等于伤痕或劳作产生的茧子，总不可能说（苏半夏和苏溯有着一段共同的经历，使他们都有了共同的泪痣）吧，这就搞不懂了…

蝴蝶

片场中有一只蝴蝶飞过，随后出现的苏半夏脖子上也有蝴蝶的样式，这tm就不得了了呀，蝴蝶在二次元领域永远暗指亡灵与轮回啊…
（别跟我说只是过场动画随意做的，这几只蝴蝶和标题上的“恋爱绮谭”上的蝴蝶是同种的，有呼应在里边）
